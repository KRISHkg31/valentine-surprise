## Method 2: GitHub Pages (Free Forever!)

### Step 1: Create GitHub Repository
1. Go to: https://github.com/new
2. Repository name: `valentine-surprise` (or any name you like)
3. Keep it **Public**
4. Click "Create repository"

### Step 2: Upload Files
1. Click "uploading an existing file"
2. Drag and drop all 3 files:
   - `index.html`
   - `style.css`
   - `script.js`
3. Scroll down and click "Commit changes"

### Step 3: Enable GitHub Pages
1. Go to **Settings** tab
2. Click **Pages** in the left sidebar
3. Under "Source", select **main** branch
4. Click **Save**
5. Wait 1-2 minutes
6. Your link will be: `https://YOUR-USERNAME.github.io/valentine-surprise`

**Pros:** Free forever, professional URL
**Cons:** Requires GitHub account
